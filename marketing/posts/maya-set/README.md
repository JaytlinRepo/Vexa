# Maya Set — 10 posts from one demo

All ten of these pieces are remixes of the single Maya interactive demo (`marketing/visuals/mockups/demo-maya/`). Same source material, ten distinct formats, ten distinct angles. The point is to prove that one product moment can become an entire week of marketing.

| # | Format | Angle | Where |
|---|---|---|---|
| 01 | Reel (animated HTML) | The morning ritual — *"Good morning. Maya is ready."* | IG Reel · Story |
| 02 | Reel (animated HTML) | The chart reveal — black past line draws, amber projection bursts to *+340%* | IG Reel · TikTok |
| 03 | Reel (animated HTML) | The clock & list — *"03:58 AM"* ticks to *04:14*, seven sources tick green | IG Reel · X video |
| 04 | TikTok script | First-person POV tap-through of the demo, "I have an analyst now" | TikTok · IG Reel |
| 05 | Carousel (6 slides) | *"How Maya works"* — walks through the demo flow as a dispatch | IG carousel |
| 06 | Carousel (8 slides) | *"The seven sources"* — one source per slide, building to "Daily." | IG carousel |
| 07 | Static image | The number — italic *7* on ivory | IG square · X · OOH |
| 08 | Static image | The button — *Approve* rendered as a sculpture | IG square · X |
| 09 | Static image | The greeting — *"Good morning."* in vertical Story | IG Story · X video card |
| 10 | X thought post | *"It's 4:14 AM."* — asymmetry as one paragraph | X / Twitter |

## How to use this set
- Stagger across 7–10 days for one launch beat.
- Reels and the TikTok are the heaviest hits — schedule those mid-week, prime time.
- Carousels work best on Tuesday/Thursday. They pull the educational reader.
- Static images and the X post are connective tissue — fill the in-between days.
- The whole set tells the *same story* (Maya finds the spike at 4 a.m., recommends Tuesday, you approve) — viewers who see two pieces feel the brand crystallizing.

## Naming convention
Each piece is in its own folder when it has assets, or a single `.md` when it doesn't. Frontmatter on every file marks `status`, `platform`, `angle`, and `priority`.
