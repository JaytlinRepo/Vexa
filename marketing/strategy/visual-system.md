# Visual System (for all marketing assets)

Pulled from the HQ v3 in-app aesthetic so marketing reads like the product. Quality bar: **Apple, Meta, big-tech flagship campaign work.** Premium minimalism. Type-as-hero. Cinematic restraint.

---

## Quality bar (read first, every time)

| Yes | No |
|---|---|
| One idea per frame | Eyebrow + hero + sub + body + quote stacked |
| ≥60% empty canvas | Edge-to-edge content |
| 1–8 words on a hero card | Paragraphs of caption-style copy |
| Type *is* the design | Type *describes* the design |
| Product floating like Apple's iPhone | Screenshots with arrows / callouts |
| Single italic amber word as focal point | Two amber words competing |
| Music + visuals + one final line (Reels) | Constant talking-head VO |

**The Apple test:** could this card hang in an Apple Store? If no, cut something.

## Palette
| Role | Hex | Use |
|---|---|---|
| Background ivory | `#fbfaf6` | Default canvas |
| Background ivory dark | `#f4f1ea` | Cards, panels, secondary surfaces |
| Ink | `#1a1a1a` | Primary type |
| Ink secondary | `#5a5856` | Body copy |
| Ink tertiary | `#8a8682` | Micro-labels, captions |
| **Accent amber** | `#c08a3e` | Hero italic word, accents, links |
| Accent soft | `#f3e7d2` | Highlight blocks |

## Type pairings
- **Display / hero:** *Cormorant Garamond, italic, regular weight.* The hero noun ("team", "CEO", "Tuesday") is in **amber** italic.
- **Body / UI:** Inter or DM Sans, 400–500.
- **Micro-labels:** JetBrains Mono, 9–11px, ALL CAPS, letter-spacing 0.18–0.22em.
- **Numbers in stats:** Cormorant Garamond, regular, with `font-feature-settings: "tnum"`.

## Layout DNA
- Hairline rules `rgba(20,16,10,.08)` separate sections only when separation is required. Not as decoration.
- Generous whitespace. Editorial gutter, never edge-to-edge content.
- One amber italic word per composition. **One.** It's the focal point.
- Numbers are big and serif. Labels above numbers are tiny and mono.

## The four allowed compositions

### A. The Word (Apple-tier hero)
- One verb or noun, italic Cormorant amber, **massive** (300–500px on a 1080 canvas)
- Dead-center vertically and horizontally
- Optional: ONE micro mono line ≤4 words, centered below
- `SOVEXA.AI` in mono at the very bottom, centered or right-aligned, 22px tracked +400
- Nothing else. No eyebrow. No hairline. No body.

### B. The Number
- Massive Cormorant italic numeral, amber. 300–500px.
- ONE label above OR below. Mono, ≤4 words.
- `SOVEXA.AI` at base. Done.

### C. The Product
- The HQ v3 dashboard (or a single in-app card) hovering in negative space
- Soft long-shadow underneath: `0 60px 120px -40px rgba(20,16,10,.20)`
- ≥40% of canvas height is empty above and below
- ONE line of caption at bottom, ink secondary
- Treat it like Apple treats an iPhone in a hero shot — composed, lit, alone

### D. The Editorial (use sparingly)
- Multi-line composition with eyebrow + hero + sub
- Reserved for thread-leads, carousel covers that intentionally read like dispatches
- Even here: still ≥50% whitespace, ≥1 hairline rule, ≤25 words total

If a piece doesn't fit A/B/C/D, it's wrong. Cut, don't add.

## Cinematic Reel grammar
- Music does the heavy lifting. Reference: classical piano, sparse cello, ambient.
- ≤6 cuts in a 20-second reel. Each cut breathes.
- Wordless until the end card. A single closing line, in italic amber.
- Product UI shown like a still-life: locked-off camera, single element, soft motion (slow zoom, parallax).

## What we never do
- Purple/blue tech gradients, glassmorphism, neon
- Screenshot collages with red circles or arrows
- AI-art faces; mid-journey aesthetics
- Emoji-heavy posts
- "🚀 Excited to announce…" ever
- Two competing focal points in one composition
- Text smaller than 16px on a 1080 canvas (illegible at thumbnail)
